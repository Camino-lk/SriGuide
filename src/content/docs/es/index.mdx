:::tip[¿Sabías qué?]
Astro te ayuda a construir sitios web más rápidos con la [“Arquitectura de Islas”](https://docs.astro.build/en/concepts/islands/).
:::

:::note
Starlight es un kit de herramientas de sitios web de documentación construido con [Astro](https://astro.build/). Puedes comenzar con este comando:

```sh
npm create astro@latest -- --template starlight
```

:::

:::caution
Si no estás seguro de querer un sitio de documentación increíble, piénsalo dos veces antes de usar [Starlight](/).
:::

:::danger
Tus usuarios pueden ser más productivos y encontrar tu producto más fácil de usar gracias a las útiles características de Starlight.

- Navegación clara
- Tema de color configurable por el usuario
- [Soporte i18n](/guides/i18n/)

:::
