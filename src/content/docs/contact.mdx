## GitHub — Bugs and Feature Requests

If you've spotted a technical issue, a broken layout, or have an idea for the site, GitHub Issues is the best place for it. It keeps things organized and lets others see if something's already been reported.

[Open an issue on GitHub →](https://github.com/Camino-lk/SriGuide/issues)

---

## What to Write About

Not sure if your message is worth sending? It probably is. Here are the kinds of things that are always welcome:

- Something on the site is outdated, wrong, or missing
- You have first-hand knowledge about a destination, experience, or service in Sri Lanka
- You'd like to contribute content or help with the site
- You want to share feedback on how the site could be more useful
- You have a question about Sri Lanka that the site didn't answer

---

## Boundaries

Above contact methods are not the right channel for the following:

- **Sponsorship enquiries** - this site doesn't accept sponsorships/paid placements
- **Urgent or emergency situations** - please see [Emergency Page](/emergency)
- **Spam, harassment, or inappropriate content**
