**Explore Sri Lanka with Confidence:** SriGuide is a free, open-source web app offering practical, up-to-date tips for travelers in Sri Lanka – built with accuracy and ease in mind.

Powered by [Astro](https://astro.build) and [Starlight](https://starlight.astro.build), SriGuide delivers a fast, multilingual, and mobile-friendly experience.

## Features

- Clean, intuitive navigation organized by key travel categories (Transport, Accommodation, Food, Attractions, Safety etc.)
- Accurate, practical information focused on real traveler needs
- Progressive Web App (PWA) support – works offline and feels like a native app
- Built for multilingual growth (i18n ready)
- Fully open-source and community-driven

## Contributing

Core content is curated and verified for accuracy, but your help is welcome and appreciated!

You can contribute in many ways:

- Report bugs or suggest improvements
- Submit or refine content (English or other languages)
- Help with translations
- Improve design, accessibility, or performance

**Got an idea? Found a bug?** Feel free to [open an issue](https://github.com/Camino-lk/SriGuide/issues/new) or submit a pull request.

**How to contribute:**

1. Fork the repository
2. Make your changes
3. Submit a pull request

It's that easy! If you're not sure how, check out GitHub's guide: [How to create a pull request in GitHub](https://opensource.com/article/19/7/create-pull-request-github).

We especially welcome your suggestions for contents and translations so this small project can reach more people.
